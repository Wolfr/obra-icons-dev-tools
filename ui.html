Production actions - Dev

Select the frame called “Obra Icons - Original source”.

Note: you might have to click this button twice to get it to work.

The dev copy serves as the base for the SVG import. Use the ID on the top right of the generated artboard to use with your SVG import code. For example pnpm run generate:icons 565:116138

Production actions - Releases - SendOwl

Select the frame called “Obra Icons - Original source” and click the button to create a design copy. This runs practically every one of the “production actions” to make a copy that can be used as a base for icons in a design system.
